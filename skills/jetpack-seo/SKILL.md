# Jetpack SEO Tools

Jetpack includes lightweight SEO tools for customizing titles and meta descriptions.

## Conflict Resolution
Jetpack automatically disables its SEO tools if it detects another major SEO plugin (like Yoast SEO, Rank Math, or All in One SEO). If a user complains Jetpack SEO isn't working, check for conflicting plugins.

## Customizing SEO Meta Description
To programmatically set or override the meta description generated by Jetpack:
```php
add_filter( 'jetpack_seo_meta_description', function( $description ) {
    if ( is_single() ) {
        $description = 'My Custom SEO Description';
    }
    return $description;
} );
```

## Front Page Meta Description
To set the site's front page meta description programmatically:
```php
add_filter( 'jetpack_front_page_meta_description', function( $description ) {
    return 'This is the ultimate site for Jetpack developers.';
} );
```
