# Jetpack VaultPress Backup for WooCommerce

This skill provides guidance on using Jetpack VaultPress Backup specifically for WooCommerce sites.

## Core Features
- Backs up all WooCommerce customer and order data.
- Restores site to any past state while keeping orders and products current.
- Custom WooCommerce table backups.
- Helps maintain GDPR compliance.

## Usage
Use this skill when dealing with eCommerce backups to ensure order integrity and safe restores without losing real-time sales data.
