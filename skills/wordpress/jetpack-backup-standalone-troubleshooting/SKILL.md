# Jetpack VaultPress Backup Troubleshooting

This skill provides information on common limitations and troubleshooting steps for Jetpack VaultPress Backup.

## Limitations
- Does NOT support WordPress multisite.
- Does NOT support split site or split home URLs.

## Troubleshooting
- If backups are missing, verify site credentials in the Jetpack Cloud Settings.
- Be aware of the `.donotbackup` file and directory exclusions if specific files are missing from the backup.

## Usage
Apply this skill when backups are failing, not running, or when assessing compatibility for complex WordPress setups.
