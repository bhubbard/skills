# Polylang Setup and Configuration

This skill provides guidance on setting up the Polylang multilingual plugin for WordPress.

## Core Setup
1. Define the default language.
2. Add additional languages.
3. Configure URL modifications (e.g., directory structures like `/en/`, `/fr/`, or distinct subdomains/domains).
4. Synchronize categories, post tags, and other metadata between translations.

## Translating Content
- Translate posts, pages, custom post types, and taxonomies by clicking the '+' icon in the Languages metabox.
- Assign languages to existing content.
- Use the Strings Translations panel for site title, tagline, and widget strings.

## Troubleshooting
- If encountering 404 errors on translated pages, flush the permalinks.
- Ensure the language code in the URL matches the configured locales.
