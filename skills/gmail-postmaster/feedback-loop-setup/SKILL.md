# Feedback Loop Setup

Use this skill when configuring or analyzing data from the Gmail Feedback Loop (FBL).

## Instructions
1. Ensure the sender is authenticated with DKIM.
2. Insert the required `Feedback-ID` header in your outgoing emails.
3. Access the FBL dashboard in Postmaster Tools to identify campaigns with high abuse complaints.
4. Recommend list hygiene and sunsetting policies based on the FBL data.
