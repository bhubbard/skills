# Elementor Theme Builder: Headers and Footers

## Overview
This skill covers the creation, styling, and conditional display of Headers and Footers using Elementor Pro's Theme Builder.

## Best Practices
1. **Semantic HTML**: Ensure the header uses `<header>` and footer uses `<footer>` HTML tags (configurable in Section/Container settings).
2. **Responsiveness**: Always optimize for Mobile, Tablet, and Desktop. Use the responsive mode in Elementor to tweak layouts, font sizes, and spacing.
3. **Display Conditions**: Carefully manage display conditions. Avoid conflicting conditions (e.g., Include Entire Site, Exclude Specific Pages).
4. **Sticky Headers**: When building sticky headers, consider using entrance animations and adjusting background colors on scroll.
5. **Dynamic Content**: Utilize dynamic tags for site logo, site title, and current year in the footer.

## Common Tasks
- Creating a transparent header that becomes solid on scroll.
- Building a custom mobile menu off-canvas or dropdown.
- Setting up a multi-level navigation menu.

## Troubleshooting
- **Missing Header/Footer**: Check display conditions and ensure no page-level templates (like Elementor Canvas) are overriding the theme builder.
- **Z-Index Issues**: If dropdown menus are hidden behind page content, increase the z-index of the header container.
