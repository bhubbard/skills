# Elementor AI Image Generation

This skill covers the capabilities of Elementor AI for generating and editing images.

## Key Features & Guidelines
1. **Prompt Creation**: Describe the subject, style, lighting, and mood clearly. (e.g., "A modern office desk with a laptop, cinematic lighting, photorealistic").
2. **Generative Fill / Edit**: Use AI to modify existing images. Select the area to replace and describe the new element.
3. **Expand Image (Outpainting)**: Extend the borders of an image naturally if it doesn't fit the desired aspect ratio.
4. **Variations**: If an image is close to what is needed, generate variations to find the perfect fit.
5. **Background Removal**: Quickly isolate subjects by removing backgrounds with a single click.
6. **Resolution**: Be mindful of the generated image resolution and use upscaling features if necessary for large displays.
