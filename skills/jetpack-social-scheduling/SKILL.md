# Jetpack Social Scheduling

This skill provides guidelines for setting up and troubleshooting scheduled posts and automatic sharing to social platforms (Facebook, Tumblr, LinkedIn, Mastodon, etc.) using Jetpack Social.

## Best Practices
- **Network Configuration**: Ensure all targeted social networks are authenticated properly in the WP Admin.
- **Scheduling Posts**: Set up a publishing schedule to maximize engagement when users are most active on social media platforms.
- **Troubleshooting**: If auto-sharing fails, check connection authorizations and review the plugin logs. Jetpack Social allows connection sharing among users who can publish posts.
