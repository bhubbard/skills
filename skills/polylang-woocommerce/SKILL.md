# Polylang for WooCommerce

This skill outlines how to use Polylang in conjunction with WooCommerce to create multilingual online stores.

## Prerequisites
- Requires the "Polylang for WooCommerce" premium add-on to bridge Polylang and WooCommerce.

## Translating Store Elements
- **Products**: Translate simple and variable products, keeping product metadata synchronized (stock, price, SKUs) across translations.
- **Categories and Attributes**: Translate product categories and global attribute terms directly within the WooCommerce interface.
- **Endpoints & Pages**: Ensure shop, checkout, cart, and 'my account' pages are properly translated and linked.

## Emails
- WooCommerce transactional emails are sent in the language the customer used when placing the order.
- Translate email strings via the Strings Translations panel.

## REST API
- Polylang Pro combined with Polylang for WooCommerce allows filtering and creating products per language via the WooCommerce REST API.
