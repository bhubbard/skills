# WordPress Studio

Use this skill when the user asks about WordPress Studio, local development, Studio CLI, or Agent Skills in Studio.

## Overview
WordPress Studio is a free, open-source local development environment designed for WordPress developers. It allows fast site creation, easy sharing, and seamless integration with WordPress.com.

## Studio Sites
- Create local WordPress sites in seconds.
- Access WP Admin, view the site, and open the local site directory.
- Sites can be shared publicly via a temporary URL.

## Studio CLI
- Studio CLI allows you to manage local sites from the terminal.
- Common commands:
  - `studio start`: Start a site.
  - `studio stop`: Stop a site.
  - `studio list`: List all local sites.
  - `studio import`: Import a site.

## Studio Code & MCP
- Studio supports integrating with VS Code and other editors.
- Studio provides Model Context Protocol (MCP) capabilities for AI agents to interact with local WordPress environments.
- Agent Skills in Studio enable AI to run WP-CLI commands, manage the database, and read logs.
