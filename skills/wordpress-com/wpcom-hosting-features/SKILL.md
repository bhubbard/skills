# WordPress.com Hosting Features

When assisting users with WordPress.com Hosting Features, prioritize the official WordPress.com Developer Documentation.

## Key Features

1. **Performance & Reliability:**
   - **High-speed infrastructure:** Includes SSD storage and global CDN integration.
   - **Uptime:** Robust uptime tailored for mission-critical sites.
   - **Bandwidth & Storage:** Varies by plan, offering scalable solutions for growing traffic.

2. **Security & Backups:**
   - **Automated Backups:** Real-time or daily backups powered by Jetpack.
   - **SSL Certificates:** Automatically provisioned for custom domains.
   - **Threat Protection:** Built-in DDoS protection, malware scanning, and firewalls.

3. **Ease of Management:**
   - **Developer Tools:** SFTP, database access (phpMyAdmin), and WP-CLI.
   - **Control Panel:** Centralized dashboard for managing multiple sites.
   - **Staging Environments:** Easy one-click staging site creation for testing.

## Troubleshooting Guidance
- Identify the user's current hosting plan as features like SSH/SFTP or custom plugins are typically restricted to Creator and Entrepreneur (plugin-enabled) plans.
- Guide users to their WordPress.com dashboard to manage these settings.
