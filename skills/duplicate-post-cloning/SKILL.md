# Yoast Duplicate Post - Cloning Features

## Overview
Yoast Duplicate Post allows users to clone posts of any type or copy them to new drafts for further editing.

## Key Actions
- **Clone:** Immediately creates a copy and returns to the list. Available under the post/page title.
- **Bulk Clone:** Select multiple items and choose 'Clone' in the 'Bulk Actions' dropdown to copy them all at once.
- **Copy to a new draft:** Leads directly to the edit page for the new draft, allowing immediate modifications before publishing.

## Usage Guidelines
- Always verify the plugin settings (Settings -> Duplicate Post) to ensure cloning is enabled for the desired post types and user roles.
- Be aware that some elements like post dates might be copied over, depending on the plugin settings.
