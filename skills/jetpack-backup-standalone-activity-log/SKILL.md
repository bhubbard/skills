# Jetpack VaultPress Backup Activity Log

This skill focuses on leveraging the Jetpack Backup activity log for precise site restorations.

## Core Features
- Complete list of all site changes.
- Identify what action (and who) broke the site.
- Quickly restore from any specific point in the log.
- On-demand "Back up now" events are also recorded here.

## Usage
When a site breaks after a specific change, use the activity log to trace the exact moment and restore precisely to the state right before the error occurred.
