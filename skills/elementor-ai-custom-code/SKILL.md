# Elementor AI Custom Code

Use Elementor AI to write and troubleshoot code within the Elementor ecosystem.

## Usage Guide
1. **Custom CSS**: 
   - Apply AI directly to widgets for custom styling.
   - Example prompt: "Make this button pulse with a soft blue glow on hover."
   - Elementor AI automatically targets the correct selector (`selector` keyword).
2. **HTML/JS Snippets**:
   - Use the HTML widget to insert AI-generated custom structure or interactive scripts.
   - Example prompt: "Create a simple countdown timer to December 31st."
3. **Code Quality**:
   - Ask Elementor AI to ensure code is responsive.
   - Request lightweight code to maintain page speed.
4. **Debugging**: If a CSS rule isn't applying, use AI to identify specificity issues or syntax errors.
