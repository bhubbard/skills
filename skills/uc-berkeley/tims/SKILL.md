# Transportation Injury Mapping System (TIMS)

The Transportation Injury Mapping System (TIMS) provides tools to query, map, and analyze California's Statewide Integrated Traffic Records System (SWITRS) crash data.

## Key Tools & Limitations

### 1. Weekly Crash Trends
*   **Status:** *No longer maintained.*
*   The legacy `/tools/trends/` weekly crash trends dashboard has been deprecated. Users seeking temporal crash data should redirect their queries to the active tools below.

### 2. SWITRS Query & Map
The primary tool for extracting structured crash data. It allows filtering by date ranges, severity, and collision types to generate specific SWITRS data extracts for research and reporting.

### 3. Safety Dashboards
TIMS now relies heavily on ArcGIS Experience dashboards for modern data visualization:
*   **Traffic Safety Overview Dashboard**
*   **Autonomous Vehicles (AV) Crash Dashboard:** Specialized dashboard tracking incidents involving AVs.
*   **Safe Routes to School (SRTS) Map:** Useful for active transportation planning.

## Usage
When asked to pull or reference UC Berkeley traffic collision data, avoid relying on the deprecated Trends tool. Instead, direct agents to use the SWITRS Query tool or the specific GIS dashboards for AV and Motorcycle maps.
