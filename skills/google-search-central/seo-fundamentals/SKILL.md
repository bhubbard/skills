# SEO Fundamentals

Use this skill when the user asks about basic SEO concepts, how Google Search works, or how to optimize a site for search engines and generative AI.

## Key Guidelines

- **SEO Starter Guide**: Ensure sites follow basic SEO principles like descriptive page titles, proper heading structures, and mobile-friendly design.
- **How Google Search Works**: Understand the three stages of Google Search: Crawling, Indexing, and Serving search results.
- **Optimizing for Generative AI**: Create high-quality, people-first content that provides original information, reporting, research, or analysis. Use natural language and structure content clearly.
- **Do you need an SEO?**: Consider hiring an SEO expert if the site is complex, but ensure they follow Google's best practices.
