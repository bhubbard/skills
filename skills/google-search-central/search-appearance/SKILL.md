# Search Appearance

Use this skill when the user wants to enhance their site's visibility and rich presentation in Google Search results.

## Key Guidelines

- **Title Links**: Provide clear, descriptive `<title>` tags for every page. Avoid keyword stuffing and generic titles like "Home" or "Profile".
- **Snippets**: Write compelling `<meta name="description">` tags that accurately summarize the page content. Google often uses these for the search snippet.
- **Structured Data**: Implement Schema.org structured data (preferably in JSON-LD format) to enable rich results like product reviews, recipes, FAQs, and events in the search gallery.
- **Images**: Optimize images by using descriptive filenames, providing `alt` text, and ensuring they are accessible to crawlers.
- **Videos**: Provide video sitemaps and use structured data for videos to help Google understand the content and display video rich results.
- **Favicons**: Provide a high-resolution favicon that represents the brand visually in search results.
