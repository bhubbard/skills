# Jetpack Social Organic Content Sharing

This skill provides instructions on maximizing engagement through Jetpack Social's Organic Content Sharing feature.

## Guidelines
- **Native Formatting**: Manually published/native content receives significantly more interaction compared to basic shared links. Use this feature to format posts natively for each platform.
- **Media Selection**: Choose specific custom images, videos, and texts tailored for native social media content.
- **Usage**: Integrate this setup in the Jetpack Social options when writing a new post in the sidebar. Select the most engaging assets rather than relying on automatic meta tags.
