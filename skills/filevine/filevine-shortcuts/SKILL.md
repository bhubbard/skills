# Filevine Shortcuts

This skill helps users learn and implement keyboard shortcuts and quick navigation features in Filevine.

## Key Areas:
- **Global Shortcuts**: Shortcuts that work anywhere in the application (e.g., quick search, creating a new task).
- **Project Navigation**: Quickly jumping between sections or tabs within a project.
- **Text Formatting**: Shortcuts for formatting text within notes or messages.
- **Action Items**: Fast ways to assign tasks, set deadlines, or change statuses.

## Guidelines for Assistance:
- Encourage the use of the quick search or command palette if available.
- Provide practical examples of how shortcuts can save time in daily workflows.
