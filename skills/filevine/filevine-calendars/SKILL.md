# Filevine Calendars Skill

This skill provides instructions for utilizing Filevine calendars to track important dates and deadlines.

## Core Capabilities

- **Event Creation**: Adding court dates, meetings, and internal deadlines.
- **Reminders**: Setting up automated reminders for approaching events.
- **Calendar Syncing**: Integrating Filevine calendars with external services (e.g., Outlook, Google Calendar).
- **Task Delegation**: Assigning specific calendar-related tasks to team members.

## Best Practices
1. Always include a location and clear description for every calendar event.
2. Utilize color-coding to distinguish between different types of events.
3. Review the upcoming week's calendar every Monday morning to anticipate workload.
