# Filevine Project Sections Skill

This skill provides guidelines and procedures for interacting with project sections in Filevine.

## Core Capabilities

- **Navigating Sections**: How to move between different project phases (e.g., Intake, Discovery, Trial).
- **Updating Data**: Best practices for entering and updating data within custom section fields.
- **Section Visibility**: Understanding permissions and visibility settings for various team roles.
- **Reporting**: Generating reports based on section-specific data points.

## Best Practices
1. Ensure all mandatory fields in a section are completed before advancing the phase.
2. Use standardized naming conventions for custom fields.
3. Regularly audit section data for accuracy and completeness.
