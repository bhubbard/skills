# Filevine Account Setup

This skill assists users in navigating the initial account setup process in Filevine. 

## Key Areas:
- **Profile Configuration**: Setting up name, contact info, and avatar.
- **Security**: Configuring password, 2FA (Two-Factor Authentication).
- **Preferences**: Adjusting timezone, language, and default landing pages.

## Guidelines for Assistance:
- Always prioritize security best practices.
- Ensure the user understands where to find their account settings (usually under their profile icon).
