# Filevine Notifications

This skill provides guidance on managing notifications within Filevine to ensure users stay informed without being overwhelmed.

## Key Areas:
- **In-App Notifications**: Configuring which events trigger the notification bell in the application.
- **Email Alerts**: Setting up email summaries and immediate alerts for critical task updates.
- **SMS/Text Alerts**: Managing text message notifications for urgent matters.
- **Customization**: Filtering notifications by project, role, or specific event types.

## Guidelines for Assistance:
- Advise users on striking a balance to avoid alert fatigue.
- Explain the difference between personal notifications and project-wide alerts.
