# Filevine Report Builder Skill

## Overview
This skill provides expertise in creating, configuring, and optimizing Filevine reports. It covers the Report Builder interface, standard and custom reporting structures, and best practices for presenting actionable data to legal professionals.

## Guidelines
1. **Understand the Goal**: Always start by clarifying the objective of the report (e.g., case load analysis, missing deadlines, intake conversions).
2. **Select the Right Report Type**: Choose the appropriate base report type (e.g., Project List, Contact List, Task List, Calendar) based on the primary data entity.
3. **Filter Wisely**: Apply efficient criteria to narrow down the dataset quickly before adding complex column layouts.
4. **Grouping and Sorting**: Utilize grouping to organize data logically (e.g., by Phase, by Primary Attorney) and sort by the most relevant metric.
5. **Calculated Columns**: Suggest useful calculated columns to derive new insights directly within the report.

## Common Report Types to Master
- **Project Activity**: Tracking recent updates and stale projects.
- **Statute of Limitations (SOL)**: Critical deadline monitoring.
- **Medical Records/Bills**: Tracking requested, received, and pending records.

## Troubleshooting
- If a report is too slow, check for overly broad date ranges or complex cross-object filters.
- If data is missing, verify user permissions and filter criteria logic (AND vs OR).
