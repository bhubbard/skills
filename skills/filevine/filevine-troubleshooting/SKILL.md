# Filevine Troubleshooting

This skill provides a structured approach for troubleshooting Filevine issues.

## General Troubleshooting Steps

1.  **Identify the Scope:**
    *   Is this affecting one user or all users?
    *   Is it specific to one project or template?
    *   Is it happening on a specific browser?

2.  **Verify Configuration:**
    *   Check user permissions.
    *   Review the Customs Editor setup for the affected section.

3.  **Clear Cache and Cookies:**
    *   Advise the user to clear their browser cache, or test in an Incognito/Private window.

## Specific Areas

*   **Reporting Issues:**
    *   Verify the report criteria (date ranges, user filters).
    *   Ensure the user running the report has permission to see the underlying data.
*   **Doc Generation:**
    *   Check for merge code errors in the Word/PDF template.
    *   Ensure the template is uploaded to the correct project template.
