# Filevine University (FVU) Training

This skill outlines how to utilize Filevine University for effective user training.

## FVU Overview

Filevine University (FVU) is the primary learning management system for Filevine users. It offers courses ranging from basic navigation to advanced administration.

## Recommended Learning Paths

1.  **New Users (General Staff):**
    *   *Filevine Basics*: Navigation, feed interaction, creating projects, and basic task management.
2.  **Project Administrators:**
    *   *Customs Editor*: Building out project templates, adding custom fields, and managing section visibility.
3.  **Reporting Leads:**
    *   *Report Builder*: Creating custom reports, scheduling reports, and managing auto-reports.

## Tracking Progress
*   Administrators should monitor course completion for new hires to ensure baseline competency.
