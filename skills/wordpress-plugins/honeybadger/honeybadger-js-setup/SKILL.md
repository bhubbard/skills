# Honeybadger JavaScript Setup

This skill assists in setting up and configuring JavaScript error monitoring using the Honeybadger WordPress plugin.

## Core Setup
1. Ensure the plugin is installed and activated.
2. Navigate to Settings -> Honeybadger in the WordPress admin panel.
3. Enter your Honeybadger JavaScript API Key. (It is recommended to have a separate Honeybadger project for JS).
4. Check the "JS error reporting enabled" option.

## Version Tracking
Use the "Version" option in the settings to specify your application's version. This helps in tracking JS errors across different application versions.

## Troubleshooting
- If errors aren't showing up, verify the JS API key in your Honeybadger account.
- Ensure "JS error reporting enabled" is checked in the plugin settings.
- Use the "Send test notification" option to verify the JS setup works, then disable it once confirmed.
