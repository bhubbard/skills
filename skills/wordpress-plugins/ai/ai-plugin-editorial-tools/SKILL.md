# Editorial Tools (WordPress AI Plugin)

The official WordPress AI plugin provides an array of opt-in editorial tools directly within the Gutenberg editor and WordPress admin. These features are modular and can be toggled on/off in `Settings -> AI`.

## Content Generation & Modification

*   **Title Generation:** Suggests post headlines. Found above the post title field; utilizes a modal to preview, regenerate, and select a title before applying it.
*   **Excerpt Generation:** Automatically creates concise summaries for posts.
*   **Content Summarization:** Summarizes long-form content into a dedicated "Content Summary" block.
*   **Content Resizing:** Allows editors to select specific block content and ask the AI to shorten, expand, or rephrase it.
*   **Content Classification:** Suggests relevant taxonomy terms (tags and categories) based on the post's content.

## SEO & Accessibility

*   **Alt Text Generation:** Generates descriptive alt text for images to improve accessibility. Accessible from the Image block inspector and the Media Library (which also supports bulk generation via UI or `wp ai alt-text generate` WP-CLI command).
*   **Meta Description Generation:** Generates SEO meta descriptions, integrating directly with popular SEO plugins.
*   **Editorial Notes:** Reviews post content block-by-block and adds actionable suggestions for Accessibility, Readability, Grammar, and SEO.
*   **Editorial Updates:** Automatically applies the suggestions generated by Editorial Notes to the content.

## Media

*   **Image Generation and Editing:** Create and edit images from prompts directly inside the post editor or the Media Library. The generated images are saved with descriptive, slugified filenames.

## Moderation & Admin

*   **Comment Moderation:** Analyzes incoming comments for toxicity and sentiment. Administrators can sort and filter the Comments screen based on these AI-generated labels to quickly identify spam or abuse.
*   **Guidelines:** Allows AI abilities to respect and follow site-wide editorial standards configured by the site administrator.
