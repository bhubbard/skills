# Google Imagen Aspect Ratios

Use this skill to create images with customized aspect ratios using Google's Imagen models in WordPress.

## Details
- Provided by the `ai-provider-for-google` plugin.
- Requires Google Gemini API key configured via `GOOGLE_API_KEY`.
