# Jetpack Search WooCommerce Integration

Jetpack Search seamlessly integrates with WooCommerce to enhance product search capabilities on eCommerce stores. Use this skill when users need help setting up or customizing search for their WooCommerce products.

## Core Features
*   **Product Filters**: Implement filters specific to WooCommerce, including Product Attribute, Price, Rating, and Stock Status.
*   **Product Categories/Tags/Brands**: Use specialized checkbox variations for filtering by product categories, tags, or brands.
*   **Product Results Layout**: Display search results optimized for products, including product image, title, price, and rating.
*   **Sorting Options**: Allow customers to sort results by Price or Rating.
*   **Product Search Overlays**: Configure the dedicated product overlay template to render product results and filters within a modal over the current page.

## Best Practices
1.  **Seamless Integration**: Jetpack Search works with WooCommerce automatically without requiring complex additional configuration for basic usage.
2.  **Filter Customization**: Leverage the Product Filters container block to group product-specific filters logically.
3.  **Real-Time Indexing**: Remind users that changes to products (price, stock status, etc.) are indexed in real-time, ensuring search results are always up to date.
4.  **Conversion Optimization**: Utilize the enhanced search capabilities to increase the likelihood of purchases, as customers can find exactly what they are looking for quickly.
