# Jetpack VideoPress Setup

## Overview
Jetpack VideoPress is an ad-free, customizable video player for WordPress that provides high-quality, lightning-fast video hosting using a global CDN.

## Features
- Provides fully-hosted videos and a CDN.
- Supports 60 FPS and 4K resolution.
- Adaptive streaming for automatic video quality adjustment.

## Setup Instructions
1. Install the Jetpack VideoPress plugin via the WordPress dashboard or download from wordpress.org.
2. Activate the plugin.
3. Configure the plugin settings to connect to the Jetpack cloud infrastructure.
4. Verify the video hosting setup by uploading a test video.
