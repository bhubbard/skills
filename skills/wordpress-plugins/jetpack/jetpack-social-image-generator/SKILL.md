# Jetpack Social Image Generator

This skill focuses on leveraging Jetpack Social's Image Generator feature to create dynamic and engaging imagery directly within the WordPress editor.

## Key Features
- **Custom Overlays**: Automatically overlay custom text onto images for social posts.
- **Styles**: Utilize a variety of styles to capture user attention and boost social engagement.
- **Workflow Optimization**: Generate images directly from the WP editor, saving time and removing the need for external image editing software.
