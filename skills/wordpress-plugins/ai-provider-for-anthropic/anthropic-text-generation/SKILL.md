# Anthropic Text Generation

Use this skill when developing or debugging text generation features in WordPress using the `ai-provider-for-anthropic` plugin and the PHP AI Client SDK.

## Capabilities
- Utilizes Claude models.
- Requires PHP 7.4+ and WordPress 6.9+ (with `wordpress/php-ai-client`) or WordPress 7.0+.
- Uses `ANTHROPIC_API_KEY` environment variable or constant.
