# Troubleshooting Mode

This skill helps you use the Health Check plugin's Troubleshooting Mode.

## Instructions
1. Go to Tools -> Site Health -> Troubleshooting.
2. Enable Troubleshooting Mode. This disables all plugins and switches to a default theme for your user session only.
3. Reactivate plugins one by one to find the source of the issue.
4. Disable Troubleshooting Mode from the admin bar when finished.
