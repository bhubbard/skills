# OpenAI Image Generation

Use this skill to generate images in WordPress using DALL-E models through the `ai-provider-for-openai` plugin and PHP AI Client SDK.

## Requirements
- OpenAI API key configured via `OPENAI_API_KEY`.
- Supports standard DALL-E image generation capabilities.
