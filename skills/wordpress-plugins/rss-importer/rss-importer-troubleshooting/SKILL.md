# RSS Importer Troubleshooting
How to resolve common errors with RSS Importer:
- **PHP 7/8 Fatal Errors**: The plugin may use deprecated functions like `set_magic_quotes_runtime`. Apply community patches or update manually.
- **Missing Content**: Check if the source feed provides full content vs excerpts.
