# Configure Akismet

This skill provides guidance on setting up Akismet for WordPress.

## Instructions
1. Upload and activate the Akismet plugin.
2. Obtain an API key from Akismet.com.
3. Enter the API key in the Akismet settings page.
4. Configure discard settings for the worst spam to save disk space.
