# Debug Akismet

This skill helps troubleshoot Akismet issues.

## Instructions
1. Check the comment status history to see if Akismet flagged the comment.
2. Review the Akismet settings to ensure the API key is active.
3. Use the "submit-spam" and "submit-ham" features to train the filter.
4. Check for conflicts with other form plugins.
