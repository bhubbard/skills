# Create SCF Fields

This skill covers creating custom fields, post types, and taxonomies with SCF.

## Instructions
1. Navigate to the SCF builder in the WordPress admin.
2. Create a new Field Group and define field types (e.g., text, repeater, gallery).
3. Set the location rules for where the fields should appear (post, page, user).
4. Use SCF functions to display the field values in theme templates.
