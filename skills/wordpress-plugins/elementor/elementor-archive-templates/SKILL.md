# Elementor Archive Templates

## Overview
This skill focuses on designing archive pages, which display lists of posts. This includes Blog Archives, Category Archives, Tag Archives, and Search Results pages.

## Key Widgets
- **Archive Title**: Dynamically displays the name of the current archive (e.g., "Category: News").
- **Archive Posts**: The main loop widget that displays the grid or list of posts belonging to the archive.

## Best Practices
1. **Custom Loops (Elementor Pro 3.8+)**: Use the Loop Builder for complete design freedom over individual post cards within the archive.
2. **Pagination**: Always include and style pagination or "Load More" buttons for archives with many posts.
3. **Empty States**: Design a friendly "Nothing Found" message for search results or empty categories.
4. **Search Form**: Include a search form on the archive page, especially on search result templates, to allow users to refine their queries.

## Advanced Techniques
- **Facet Filtering**: Integrating with plugins like FacetWP or WP Grid Builder for advanced filtering on archive templates.
- **Custom Queries**: Using the query controls in the Archive Posts widget to exclude certain categories or include specific post types.
