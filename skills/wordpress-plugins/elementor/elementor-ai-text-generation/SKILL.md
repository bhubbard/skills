# Elementor AI Text Generation

This skill guides the use of Elementor AI for text-related tasks. 

## Best Practices
1. **Context is Key**: Always provide Elementor AI with the context of the website, brand voice, and target audience.
2. **Iterative Refinement**: Use the tools to make text shorter, longer, or change tone (e.g., professional, casual).
3. **Translation**: Use the built-in translation feature to adapt content for global audiences easily.
4. **Prompt Engineering**: Be specific about the desired output format (e.g., bullet points, paragraphs, catchy headlines).
5. **Review**: Always review AI-generated text for accuracy and brand alignment before publishing.
