# View Transitions Exclusions

- If a theme already implements view transitions, the plugin allows users to override it via settings.
- The plugin handles its own logic to not double-apply transitions.
- Exclusions can also apply via browser-level support or if core already implements it (the plugin omits functionality if core has it).
