# Yoast SEO XML Sitemaps

Yoast SEO automatically generates an XML sitemap for the site. Developers often need to hook into this process to include custom URLs, exclude specific posts, or add new sitemap indexes.

## Reference
[XML Sitemaps Documentation](https://developer.yoast.com/features/xml-sitemaps/)

## Common Filters

### Exclude a specific post
You can exclude a post from the sitemap using the `wpseo_exclude_from_sitemap_by_post_ids` filter.
```php
add_filter( 'wpseo_exclude_from_sitemap_by_post_ids', 'exclude_post_from_sitemap' );
function exclude_post_from_sitemap( $excluded_posts_ids ) {
    $excluded_posts_ids[] = 42; // Exclude post ID 42
    return $excluded_posts_ids;
}
```

### Change the number of entries per sitemap
By default, Yoast outputs up to 1000 URLs per sitemap page. You can alter this to prevent memory limits on huge sites.
```php
add_filter( 'wpseo_sitemap_entries_per_page', 'max_sitemap_entries' );
function max_sitemap_entries( $entries ) {
    return 500;
}
```

### Adding custom URLs to the Sitemap
If you have a custom route or a hardcoded page not stored in WordPress posts/pages, you can hook into a specific sitemap (like the page sitemap) to append items.

```php
add_filter( 'wpseo_sitemap_page_content', 'add_custom_url_to_sitemap' );
function add_custom_url_to_sitemap( $content ) {
    $url = 'https://example.com/my-custom-route/';
    $date = date( 'c' );
    
    $content .= "<url>\n";
    $content .= "<loc>{$url}</loc>\n";
    $content .= "<lastmod>{$date}</lastmod>\n";
    $content .= "</url>\n";
    
    return $content;
}
```

## Best Practices
- If you add custom URLs using `wpseo_sitemap_page_content`, you are appending literal XML strings. Ensure your XML syntax is valid.
- If a user reports the sitemap is 404ing or not updating, the most common fix is flushing WordPress rewrite rules (Settings > Permalinks > Save Changes) or clearing server-level caching. Yoast sitemaps are generated on the fly and are not static XML files.
