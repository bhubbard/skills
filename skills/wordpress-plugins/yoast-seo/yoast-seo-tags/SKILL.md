# Yoast SEO Tags

Yoast SEO takes over the generation of the HTML `<title>`, `<meta name="description">`, `<link rel="canonical">`, and `<meta name="robots">` tags.

## Reference
[SEO Tags Documentation](https://developer.yoast.com/features/seo-tags/canonical-urls/)

## Key Filters

### Title Tag (`wpseo_title`)
Allows you to completely overwrite the `<title>` tag generated by Yoast.
```php
add_filter( 'wpseo_title', 'modify_yoast_title' );
function modify_yoast_title( $title ) {
    if ( is_singular( 'product' ) ) {
        return 'Buy ' . get_the_title() . ' Online - My Store';
    }
    return $title;
}
```

### Meta Description (`wpseo_metadesc`)
Allows you to overwrite the `<meta name="description">`.
```php
add_filter( 'wpseo_metadesc', 'modify_yoast_metadesc' );
function modify_yoast_metadesc( $description ) {
    if ( is_category() ) {
        return 'Custom description for ' . single_cat_title( '', false );
    }
    return $description;
}
```

### Canonical URL (`wpseo_canonical`)
Allows you to change the URL output in the `<link rel="canonical">` tag.
```php
add_filter( 'wpseo_canonical', 'modify_yoast_canonical' );
function modify_yoast_canonical( $canonical ) {
    // Force the canonical to a specific URL based on custom logic
    if ( is_page( 'landing-page-a' ) ) {
        return 'https://example.com/main-landing-page/';
    }
    return $canonical;
}
```

### Meta Robots (`wpseo_robots`)
Allows you to adjust the indexing instructions (e.g., changing `index` to `noindex`).
```php
add_filter( 'wpseo_robots', 'modify_yoast_robots' );
function modify_yoast_robots( $robots ) {
    // If a custom condition is met, enforce noindex, nofollow
    if ( is_singular() && get_post_meta( get_the_ID(), '_my_custom_hidden_flag', true ) ) {
        return 'noindex, nofollow';
    }
    return $robots;
}
```

## Best Practices
- When using `wpseo_title`, return the full, raw string. Yoast will handle escaping it before injecting it into the HTML `<title>`.
- Do not use `add_theme_support('title-tag')` filters (like `document_title_parts`) if Yoast is active, as Yoast completely bypasses standard WordPress title generation for better performance and consistency. Use `wpseo_title` instead.
