# Export Theme Changes

The Create Block Theme plugin allows you to save user-made customizations (styles, templates, fonts) directly back into the theme files.

## Workflow
1. Use the WordPress Site Editor to modify styles, fonts, and templates.
2. Click the Create Block Theme icon (next to the Save button).
3. Select **Save Changes** to overwrite the active theme's files with your new customizations.
4. To export, select **Export Zip** to download a finalized, export-ready version of your theme.

The plugin automatically processes block markup to be export-ready and ensures images used in templates are moved to the theme's `assets` folder.
