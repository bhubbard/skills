# Create Child Theme

Creating a child block theme allows you to inherit the parent theme's functionality while applying custom modifications.

## Steps
1. Navigate to the Create Block Theme panel in the WordPress Site Editor.
2. Click **Create Theme**.
3. Select **Create Child Theme**.
4. Provide a name and optional metadata for your new child theme.
5. The plugin will generate and activate the child theme, preserving your current settings and configurations.
