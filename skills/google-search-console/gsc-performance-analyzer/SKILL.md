# Google Search Console Performance Analyzer

Use this skill when the user needs to analyze their Google Search Console Performance report data.

## Core Metrics

*   **Clicks**: Number of times users clicked from Search results.
*   **Impressions**: Number of times a link was seen in Search results.
*   **CTR (Click-Through Rate)**: Clicks divided by Impressions.
*   **Average Position**: Average ranking in search results for a given query.

## Analysis Workflows

1.  **Low-Hanging Fruit Optimization**:
    *   Find queries where the Average Position is between 11 and 20 (Page 2 of results).
    *   Identify queries with High Impressions but Low CTR (often position 1-5). Suggest improving title tags and meta descriptions to entice more clicks.
2.  **Content Gap Analysis**:
    *   Identify queries the site ranks for but doesn't have a dedicated page for. Suggest creating targeted content.
3.  **Performance Drops**:
    *   If traffic dropped, compare date ranges (e.g., last 28 days vs. previous 28 days).
    *   Isolate drops by Query, Page, Device, or Country to pinpoint the cause (e.g., algorithmic update vs. technical issue on a specific page).
